# News Analysis & Outcome Analyzer

AI-powered News Analysis & Outcome Prediction System using RAG (Retrieval-Augmented Generation), transformer-based NLP models, sentiment analysis, contextual news retrieval, and outcome forecasting for intelligent news understanding and prediction.

---

## Features

- News Retrieval & Processing
- Sentiment Analysis
- Entity Extraction
- Retrieval-Augmented Generation (RAG)
- Contextual News Understanding
- Outcome Prediction
- LLM-based Explanations
- Historical News Ingestion
- Vector Database Integration using ChromaDB

---

## Tech Stack

- Python
- Transformers / HuggingFace
- DistilBERT
- ChromaDB
- RAG Pipeline
- NLP
- PyTorch
